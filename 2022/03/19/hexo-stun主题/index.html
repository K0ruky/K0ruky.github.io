个人博客新手Mac搭建实操

发表于2022-03-19更新于2022-03-20字数统计791阅读时长5分阅读次数

准备GitHub账号并新建一个博客仓库（Gitee搭建点击这里）

仓库命名必须为username.github.io且为公开状态
eg.

准备工具

homebrew安装vscode

homebrew install --cask visual-studio-code

部署hexo框架

node.js安装

brew install node

hexo框架安装

npm install -g hexo-cli

新建博客文件夹并初始化hexo

mkdir blog && hexo init

修改hexo目录下配置文件

vi _config.yml

将示例图中repository修改为你的GitHub博客仓库即可。

安装hexo推送插件并推送至GitHub

npm install hexo-deployer-git --save && hexo clean && hexo g && hexo d

至此hexo博客搭建完成，访问username.github.io即可（username为你的GitHub用户名）

stun主题安装

git clone https://github.com/liuyib/hexo-theme-stun.git themes/stun

修改网站标题等信息

终端进入博客文件夹执行命令使用vscode打开项目

code ./

修改bolg目录下的_config.yml文件

title、subtitle、author、language、url

theme修改为stun

每次修改bolg配置文件后可在mac终端中bolg目录下执行hexo s本地部署，访问http://localhost:4000 查看修改效果，下同

自用stun主题配置

终端执行新增首页分类、标签按钮命令

hexo new page tags && hexo new page categories

vscode编辑source目录下对应文件夹中的index.md，加上type字段，如：

去除blog/themes/stun/下的_config.yml注释。

更改博客首页图片信息

首先在vscode中为blog/source/下新增一个图片存放目录stun_images并将需要使用的图片拖到改目录下；修改blog/themes/stun/下的_config.yml文件，ctrl+f搜索bg_image并设置为true，url中填写图片位置即可。

修改个人头像

上述**_config.yml文件，ctrl+f搜索author并设置为true，url中填写图片位置即可，motto**自行修改为想要的内容。

添加网站主体特效

上述**_config.yml文件，ctrl+f搜索canvas_ribbon和canvas_nest依旧个人喜好设置为true**开启并设置特效幅度参数即可。

添加右下角2d玩偶

npm install --save live2d-widget-model-hijiki && npm install --save live2d-widget-model-hijiki

在blog/修改yml文件，新增如下内容：
live2d:
enable: true # 是否启动
scriptFrom: local # 默认
pluginRootPath: live2dw/ # 插件在站点上的根目录(相对路径)
pluginJsPath: lib/ # 脚本文件相对与插件根目录路径
pluginModelPath: assets/ # 模型文件相对与插件根目录路径
tagMode: false # 标签模式, 是否仅替换 live2d tag标签而非插入到所有页面中
debug: false # 调试, 是否在控制台输出日志
model:
use: live2d-widget-model-hijiki ## 模型文件
display:
position: right # 定位方向 left right top bottom
width: 150 # 小人宽度
height: 300 # 小人高度
hOffset: -15 # 向 偏移
vOffset: -15 # 像 偏移
mobile:
show: true # 手机端是否显示
react:
opacity: 1.0 # 模型透明度

至此博客的基本美化配置已完成，保存修改后终端执行hexo s查看部署效果，执行hexo clean && hexo g && hexo d 部署至GitHub后可以通过username.github.io进行访问

首先需要注册GitHubApplication授权获取client_id和client_secret
修改blog/themes/stun/_config.yml文件即可。
